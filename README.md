# Zig Wasm Pong Game

A simple Pong game running through **WebAssembly (WASM)** and rendered on an HTML `<canvas>`.  
Built using **Zig** and **Raylib**.

---

## Where to play

You can play this game on [pong.kwameab.com](https://pong.kwameab.com/).

## How to play locally

To run this locally `git clone` this repo and run

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080
